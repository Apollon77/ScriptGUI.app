# ScriptGUI.app
## Version 0.7.?

CCU-IO.ScriptGUI ist ein Visueller Script builder auf Node-Webkit basis. Mit ihm dem es möglich ist Scripte für CCU.IO / ioBroker zu erstellen, zu bearbeiten und zu testen.
